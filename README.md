# ldb-rs
Motor de almacenamiento NoSQL embebido desarrollado desde cero en Rust. Diseñado como referencia arquitectónica de sistemas, implementa serialización binaria compacta empaquetada, manejo seguro de punteros mediante una arena de nodos para el B+ Tree, resiliencia con checksums CRC64 en sus cabeceras y portabilidad completa entre plataformas.
